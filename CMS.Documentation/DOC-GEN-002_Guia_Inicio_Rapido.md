## 🚀 Primeros Pasos en 5 Minutos

Esta guía le ayudará a comenzar a usar el sistema CMS rápidamente.

---

## Paso 1: Acceder al Sistema

### 1.1 Abrir el navegador

Ingrese a: **https://cms.biti-solutions.com**

### 1.2 Seleccionar Compañía

```
┌─────────────────────────────────────┐
│  Ingrese el código de su compañía  │
│                                     │
│  Código: [sinai____________]        │
│                                     │
│  [      Continuar      ]            │
└─────────────────────────────────────┘
```

Ingrese el código proporcionado por su administrador (ej: `sinai`, `eamr`).

### 1.3 Iniciar Sesión

**Opción A - Azure AD:**
- Clic en "Iniciar con Microsoft"
- Use sus credenciales corporativas

**Opción B - Login Local:**
- Email: su_email@empresa.com
- Contraseña: la proporcionada

## Paso 3: Navegación Básica

### Menú Lateral

El menú a la izquierda muestra los módulos disponibles según sus permisos:

1. **Hacer clic** en un módulo para expandirlo
2. **Hacer clic** en una opción para acceder
3. **El ícono ◀** colapsa/expande el menú

### Acciones Comunes

| Acción | Cómo hacerlo |
|--------|--------------|
| Buscar | Usar el campo de búsqueda en cada lista |
| Crear nuevo | Botón "➕ Nuevo" o "Agregar" |
| Editar | Clic en el ícono ✏️ o en la fila |
| Eliminar | Clic en el ícono 🗑️ (requiere confirmación) |
| Exportar | Botones Excel, PDF, CSV donde disponibles |

---

## Paso 4: Módulos Frecuentes

### 📦 Inventario

Para ver artículos:
1. Menú → Inventory → Items
2. Use los filtros para buscar
3. Clic en un artículo para ver detalles

### 📈 Reportes

Para ejecutar un reporte:
1. Menú → Reports & BI → General
2. Seleccione el reporte deseado
3. Configure los filtros
4. Clic en "Ejecutar Reporte"
5. Exporte si necesita (Excel/PDF/CSV)

### 🔐 Administración

Para gestionar usuarios:
1. Menú → Administration → Users
2. Clic en "➕ Nuevo Usuario" para crear
3. Clic en un usuario para editar permisos

---

## Paso 5: Cerrar Sesión

Para cerrar sesión de forma segura:

1. Clic en su nombre de usuario (esquina superior derecha)
2. Seleccione "Cerrar Sesión"
3. Será redirigido a la pantalla de selección de compañía

> ⚠️ **Importante:** Siempre cierre sesión al terminar, especialmente en equipos compartidos.
